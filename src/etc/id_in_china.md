# 中华人民共和国身份证号码组成

以下内容皆来自百度，当作数据处理时可以使用的知识。

- 由18位数字组成
- 第1,2位数字表示：所在省份代码
    - 11-15：京、津、冀、晋、蒙
    - 21-23：辽、吉、黑
    - 31-37：沪、苏、浙、皖、闽、赣、鲁
    - 41-46：豫、鄂、湘、粤、桂、琼
    - 50-54：渝、川、贵、云、藏
    - 61-65：陕、甘、青、宁、新
- 第3,4位数字表示：所在城市代码
    - 01-20,51-70表示地级市
    - 21-50表示地区（自治州、盟）
- 第5,6位数字表示：所在区县代码
    - 01-18表示地级市、自治州、地区、盟辖县级市
    - 21-80表示县（旗）
    - 81-99表示省直辖县级行政单位
- 第7-14位数字表示：出生日期（格式：yyyymmdd）
- 第15-17位数字都是统一地址辖区内的，以及同年同月同日出生人的顺序码，同时兼具性别标识功能，男单女双
    - 理论上在 $60*2 = 120$ 以内
- 第18位数字是校验码：可以是0-9和X，其中X代表10
    - 计算公式：$(12 - \Sigma{(A_i * W_i) \mod{11}}) \mod{11}$
    - i 表示身份证号码从左至右的位置序号，取值1-17
    - $A_i$ —表示第i位上的数字；Wi—表示第i位上的加权因子
    - 加权因子计算公式：$W_i = 2^{i-1}\mod{11}$
    - 校验码计算结果为0-10之一，其中“10”用“X”表示。
